# SIGGIP – Sistema Web de Gestión de Prácticas Profesionales

Sistema web basado en arquitectura de tres capas para gestionar publicación de prácticas, postulaciones, seguimiento, evaluaciones y métricas.

## Tecnologías
- **Frontend:** React 18, TailwindCSS, Vite
- **Backend:** Node.js, Express.js, Sequelize, JWT, Bcrypt
- **Base de Datos:** PostgreSQL 14
- **Automatización:** n8n (notificaciones)
- **Testing:** Jest, Cypress
- **Control de Versiones:** Git / GitHub
